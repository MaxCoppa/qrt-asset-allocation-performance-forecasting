# Tree-Based Models

This module implements models based on decision trees and their ensembles for asset allocation forecasting.

## Components
- **Models**:  
  Initialization of models used in the Project.
- **Evaluation**:  
  Metrics for model assessment.
- **Selection**:  
  Utilities for comparing models under cross-validation.
- **Tuning**:  
  Hyperparameter optimization methods.
